Obtenir une clé Groq

Gratuit, ~2 min, sans carte bancaire.

  1. Ouvre console.groq.com/keys et connecte-toi (un compte Google suffit).
  2. Clique sur Create API Key, donne-lui un nom, valide.
  3. Copie la clé affichée (elle commence par gsk_…).
  4. Reviens ici, colle-la dans Clé API Groq, puis Enregistrer.

La clé reste sur ton appareil et n'est envoyée qu'à Groq.
